# 情绪分析框架

## 分析维度

### 1. 机构持仓分析

**关键指标**：
| 指标 | 含义 | 解读 |
|------|------|------|
| 机构持仓比例 | 机构持有股份占总股本 | >50% 机构认可度高 |
| 持仓变化 | 季度增减持情况 | 持续增持/减持信号 |
| 前十大股东 | 主要持股机构 | 长期资金 vs 短期资金 |
| 北向/外资持仓 | 海外资金持股 | 反映外资偏好 |

**分析要点**：
- 机构持仓是增加还是减少
- 新进/退出的机构类型（公募、私募、社保、QFII）
- 持股周期（长期配置型 vs 交易型）

### 2. 分析师预期

**一致预期数据**：
- 目标价（均值、中位数、最高、最低）
- 评级分布（买入/增持/中性/减持）
- 盈利预测调整趋势

**评级信号**：
| 信号 | 含义 |
|------|------|
| 上调评级 | 利好信号 |
| 下调评级 | 利空信号 |
| 目标价上调 | 正向调整 |
| 盈利预测上调 | 预期改善 |

### 3. 市场情绪指标

**技术情绪**：
- 成交量变化（放量/缩量）
- 股价相对强弱（RS）
- 资金流向（主力净流入）

**舆情情绪**：
- 新闻情感分析
- 社交媒体讨论热度
- 研报覆盖数量变化

### 4. 估值情绪

**市场预期**：
- 当前股价隐含的增长预期
- 与历史估值中枢的偏离
- 市场一致预期 vs 实际表现

## 分析模板

```markdown
## 情绪分析

### 机构持仓
**评分：[1-5星]**

核心发现：
1. 机构持仓比例 [X]%，较上季度 [变化]
2. 主要机构：[列出知名机构]
3. 持仓变化趋势：[增持/减持/稳定]
4. 外资持仓：[X]%，[增加/减少]

数据摘要：
| 指标 | 数值 | 变化 |
|------|------|------|
| 机构持仓占比 | XX% | +Xpp |
| 基金数量 | XX | +X |
| 北向持股 | XX亿 | +X% |

### 分析师预期
**评分：[1-5星]**

核心发现：
1. 一致目标价 [X] 元，当前价格 [X] 元，上涨空间 [X]%
2. 评级分布：买入 [X]%，增持 [X]%，中性 [X]%
3. 盈利预测：2025E EPS [X]，同比 [X]%
4. 近期调整：[上调/下调] 评级 [X] 次

### 市场情绪
**评分：[1-5星]**

核心发现：
1. 近期成交量：[放量XX%/缩量]
2. 资金流向：[主力净流入/流出 X亿]
3. 股价相对强弱：[强于/弱于] 大盘
4. 舆论热度：[高/中/低]

### 情绪综合结论
[2-3 句话总结市场情绪状态]

**当前市场可能定价了 [正面/中性/过度乐观/过度悲观] 的预期**
```

## 数据来源

1. **机构持仓**：使用 institutional-holdings skill
2. **分析师评级**：使用 stock-deep-dive skill
3. **资金流向**：使用 northbound-flow 或 finance-data-retrieval
4. **舆情数据**：WebSearch 获取近期新闻
