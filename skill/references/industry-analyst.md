# 行业分析框架

## 分析维度

### 1. 行业空间分析

**市场测算**：
| 指标 | 说明 |
|------|------|
| 行业规模 | 当前总市场容量（GMV/产值） |
| 增速 | 行业增长率（CAGR） |
| 渗透率 | 当前渗透率 vs 理论上限 |
| 增长驱动 | 需求端 + 供给端推动因素 |

**生命周期阶段**：
```
导入期 → 成长期 → 成熟期 → 衰退期
  ↑         ↑         ↑
高增速   渗透率提升  增速放缓
高不确定  竞争加剧    格局稳定
```

### 2. 竞争格局分析

**波特五力分析**：

| 力量 | 分析要点 | 强度评估 |
|------|----------|----------|
| 行业内竞争 | 竞争者数量、集中度、差异化 | 高/中/低 |
| 新进入者威胁 | 壁垒（资本、技术、品牌、许可） | 高/中/低 |
| 替代品威胁 | 替代品性价比、用户转换成本 | 高/中/低 |
| 上游议价 | 供应商集中度、可替代性 | 强/中/弱 |
| 下游议价 | 客户集中度、信息透明度 | 强/中/弱 |

**竞争格局类型**：
| 类型 | 特征 | 投资逻辑 |
|------|------|----------|
| 完全竞争 | 分散，无数小玩家 | 难选股，看龙头集中度提升 |
| 寡头垄断 | 2-5 家主导 | 关注龙头，警惕价格战 |
| 双寡头 | 2 家主导 | "双寡头"格局最稳定 |
| 垄断 | 一家独大 | 享受定价权，关注政策风险 |

### 3. 产业链位置分析

**价值链分析**：
```
原材料 → 零部件 → 整机制造 → 渠道 → 品牌 → 用户
  ↓         ↓         ↓        ↓      ↓
 上游弱    中游制造   中游     下游    最强
议价      差异化难   差异化    渠道为王 定价权
```

**关键环节判断**：
- 哪个环节占据最多利润
- 哪个环节壁垒最高
- 哪个环节增速最快

### 4. 政策环境分析

**政策影响维度**：
| 政策类型 | 影响方向 | 典型案例 |
|----------|----------|----------|
| 行业监管 | 规范/限制 | 互联网反垄断、教育双减 |
| 财政补贴 | 支持/激励 | 新能源补贴、半导体扶持 |
| 税收优惠 | 降低成本 | 高新技术企业税收优惠 |
| 贸易政策 | 进出口影响 | 关税调整、出口管制 |
| 环保政策 | 成本/产能 | 双碳目标、限产 |

## 分析模板

```markdown
## 行业分析

### 行业空间
**评级：[好/中/差]**

核心发现：
1. 行业总规模 [X] 亿，预计 2027 年达 [X] 亿
2. 行业增速 [X]%（CAGR），[加速/放缓] 中
3. 当前渗透率 [X]%，[提升/下降] 空间 [大/中/小]
4. 主要增长驱动：[列举 2-3 个]

### 竞争格局
**格局类型：[X]**

核心发现：
1. 主要玩家：[列出 TOP 3-5 公司及份额]
2. 竞争强度：[激烈/一般/温和]
3. 公司市场地位：第 [X] 名，份额 [X]%
4. 核心竞争要素：[价格/技术/渠道/品牌]

波特五力评估：
| 力量 | 强度 | 关键因素 |
|------|------|----------|
| 行业内竞争 | 高 | ... |
| 新进入者 | 中 | 壁垒是... |
| 替代品 | 低 | ... |
| 上游议价 | 弱 | ... |
| 下游议价 | 中 | ... |

### 产业链位置
**公司位于：[X] 环节**

价值链分布：
- 利润分布：[上游/中游/下游] 最赚钱
- 公司定位：[制造/品牌/渠道]
- 核心优势：[列举]

### 政策环境
**政策支持度：[强支持/中性/受限]**

核心政策影响：
1. [政策名称]：[正面/负面] 影响 [具体方面]
2. 政策风险：[高/中/低]
3. 应对能力：[公司如何应对]

### 行业综合结论
[2-3 句话总结行业赛道质量]

**赛道评级：[上等/中上/中等/中下/下等]**
```
