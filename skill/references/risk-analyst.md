# 风险评估框架

## 风险分类体系

### 1. 经营风险（Company-Specific）

| 风险类型 | 评估要点 | 红旗信号 |
|----------|----------|----------|
| 商业模式 | 盈利模式是否可持续 | 过度依赖单一客户/产品 |
| 管理层 | 诚信、能力、稳定性 | 频繁换帅、大股东减持 |
| 经营效率 | 资产周转、运营能力 | 存货/应收账款异常增长 |
| 增长质量 | 内生增长 vs 并购增长 | 商誉减值风险 |

### 2. 财务风险（Financial）

| 风险类型 | 评估指标 | 警戒线 |
|----------|----------|--------|
| 流动性风险 | 流动比率、速动比率 | < 1.0 |
| 偿债风险 | 资产负债率、有息负债/EBITDA | > 70%、> 5x |
| 现金流风险 | 经营现金流/净利润 | < 0.8 |
| 会计风险 | 应收账款周转、存货周转 | 持续恶化 |

### 3. 市场风险（Market）

| 风险类型 | 评估要点 | 量化指标 |
|----------|----------|----------|
| 估值风险 | 当前估值处于历史高位 | PE/PB > 历史 90% |
| 筹码风险 | 机构持股集中度 | 前十大股东 > 80% |
| 流动性风险 | 日均成交量、流通盘比例 | 成交量持续萎缩 |
| 系统性风险 | 与大盘相关性 | Beta > 1.5 |

### 4. 外部风险（External）

| 风险类型 | 评估要点 | 监测指标 |
|----------|----------|----------|
| 政策风险 | 行业监管走向 | 政策变化跟踪 |
| 宏观风险 | 经济周期、利率环境 | PMI、CPI、利率 |
| 汇率风险 | 出口占比、海外收入 | 汇率敏感度 |
| 竞争风险 | 新进入者、替代品 | 市场份额变化 |

## 风险评估矩阵

```
严重性
  ↑
  │  ●关注   ●高危
  │  □监控   ■极高危
  │  □低      ■
  └─────────────────→ 可能性
```

**评估标准**：
- **极高危**：高可能性 + 高严重性 → 必须规避
- **高危**：中可能性 + 高严重性，或高可能性 + 中严重性 → 谨慎
- **关注**：低可能性 + 高严重性，或中可能性 + 中严重性 → 监控
- **低**：低可能性 + 低严重性 → 可接受
